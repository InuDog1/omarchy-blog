Omarchy の最新アップデート「The Defaults Edition (v3.8.0)」がリリースされました！
前回のアップデート3.7Releaseが5日前なのでとても速いですね！
今回のアップデートでは、多数の新機能、設定オプションの拡充、外観のカスタマイズ性の向上など、より快適なデスクトップ環境を実現するための改善が行われています。

## 📥 アップデート・インストール方法

- **既存ユーザー:** Omarchyメニュー（`Super + Alt + Space`）から **[Update] > [Omarchy]** を選択してアップデートできます。
- **新規インストール（ISOダウンロード）:**
  - **Download:** [https://iso.omarchy.org/omarchy-3.8.0.iso](https://iso.omarchy.org/omarchy-3.8.0.iso)
  - **SHA256:** `a9b271e8884dc123f0787e10cff3153dca7361cc6583edd1a3baa283a031f872`

---

## ✨ 新機能 (Features)

- **リマインダー機能の追加:** `Super + Ctrl + R`で設定、`Super + Ctrl + Alt + R`で一覧表示、`Super + Ctrl + Shift + R`で全クリアが可能。メニューの `Trigger > Reminder` からもアクセスできます。
- **ライブ天気表示:** Waybar上や、ショートカット（`Super + Ctrl + Alt + W`）の通知で現在の天気を確認できるようになりました。
- **メディアのトランスコード:** 動画、音声、GIFのトランスコード機能を追加。メニューの `Trigger > Transcode` や `Super + Ctrl + .`、CLIから実行できるほか、ファイルマネージャーの右クリックメニューにも追加されました。
- **RetroArchの強化:** デフォルト設定に Amiga、C64、Amstrad のコアを追加しました。
- **ディレクトリ形式のフック機能:** 単一ファイルではなく `~/.config/omarchy/hooks/post-update.d/` のようなディレクトリ形式のフックをサポートし、拡張機能の管理が容易になりました。起動後の通知フックも追加されています。
- **Voxtype Dictation:** 初回アップデート後、新規インストール環境に対して音声入力ツールのインストールを提案するようになりました。
- **Bluetooth接続の安定性向上:** WirePlumberにBluetooth A2DPの自動接続機能を追加しました。

## ⚙️ デフォルト設定・インストール (Defaults)

- **デフォルトアプリの選択:** `Setup > Default > Browser/Terminal/Editor` から、システム全体のデフォルトアプリケーションを簡単に設定できるようになりました。
- **ブラウザの簡単インストール:** `Install > Browser` から、Chrome、Brave、Brave Origin、Firefox、Zen をインストールし、デフォルトに切り替えることができます。
- **ターミナルオプションの追加:** インストール可能なターミナルとして、ライブテーマ機能を持つ「Foot」が選べるようになりました。
- **ISOインストーラーの改善:** 暗号化なしでのインストールオプションや、Colemakキーボードレイアウトのオプションが追加されました。

## 🎨 外観・テーマ (Aesthetics)

- **ASCIIロゴ生成:** 任意の画像からASCIIロゴのスクリーンセーバーやAbout画面を生成する機能を追加しました。
- **UIの一貫性とテーマ対応:**
  - `Install > Editor` 経由でインストールされた Zed エディタのテーマに対応。
  - SDDMログイン画面のテーマが、復号化画面のスタイルと完全に一致するようになりました。
  - pi agent のライト/ダークモード同期をサポートしました。
- **テーマ関連の修正:**
  - Catppuccin テーマのカラースキーム名を修正。
  - テーマ更新時に、選択した背景画像が意図せず変更される問題を修正。
  - 背景のシンボリックリンク作成前の相対壁紙パスの問題を修正。
  - ノーギャップ（隙間なし）モードを有効にした際、ウィンドウの角丸も自動的に無効化されるよう修正。
  - Helix エディタのテーマの視認性を改善。

## ⌨️ 操作・コントロール (Controls)

- **アイドル処理の改善:** ロック/スリープ/復帰時の処理が見直され、ディスプレイとキーボードのバックライトのオフ・オンがより確実に動作するようになりました。
- **中クリックでのペーストの統一:** マウスのボタン3（中クリック）によるプライマリ選択の貼り付けを有効にし、GTKアプリケーションとChromium系アプリケーション間での一貫性を向上させました。
- **CLI・ターミナルの操作性向上:** 
  - tmuxの拡張キーをデフォルトで追加。
  - Omarchyのコマンド補完が尽きた後に、ファイルの補完が機能するようになりました。

## 💻 ハードウェア互換性 (Compatibility)

- **Studio Display XDR** のサポートを追加。
- **Dell XPS 2026** 向けに、タッチパッドのハプティック強度を調整できる設定（`Trigger > Hardware > Touchpad Haptic`）を追加。
- **Intel Panther Lake (DSP搭載機)** におけるオーディオの問題を sof-firmware のインストールにより修正。
- **Xシリーズ以外の Panther Lake チップ** におけるハードウェア・ビデオ・アクセラレーションを修正。
- **Lenovo Yoga Pro 7 14IAH10** のバススピーカーの不具合を修正。
- **Asus Zenbook UX5406AA** のディスプレイ・バックライトの不具合を修正。

## 🔧 その他の修正・改善 (Fixes)

- 画面を手動でロックした際、直後にディスプレイの電源が正しくオフになるよう修正。
- RetroArch のジョイパッド自動設定とディレクトリのセットアップ問題を修正。
- Omarchy CLI のメタデータの不整合を修正。
- `omarchy refresh applications` 実行時に npx のスタブも正しく更新されるよう修正。
- 一部の既存 Linux 環境において、インストーラーのフォーマット処理が破損する問題を修正。
- piパッケージが `@earendil-works/pi-coding-agent` に移行したことに対する修正。
- 同一の通知が適切にグループ化されるよう修正。
